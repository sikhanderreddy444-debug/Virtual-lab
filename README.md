# Virtual ECE Lab: Analog and Linear Electronic Circuits

A digital laboratory platform for ECE students to learn theory, perform virtual experiments, visualize circuit behavior, and analyze data. 

## 🚀 Features
- **Interactive Simulations:** Change input values and see real-time output changes.
- **Data Visualization:** Auto-generated V-I characteristic graphs using Chart.js.
- **Academic Structure:** Includes Theory, Circuit Diagrams, Procedures, and Observation Tables.
- **Client-Side Processing:** Extremely fast, requires no backend, easily hosted on GitHub Pages.

## 🛠️ Technology Stack
- **Frontend:** HTML5, CSS3, JavaScript
- **Styling:** Bootstrap 5
- **Graphing:** Chart.js

## 🧫 Experiments Included (Prototype)
1. PN Junction Diode V-I Characteristics
*More to be added: Rectifiers, BJT, Op-Amps, Filters.*

## 💻 How to Run Locally
1. Clone the repository: `git clone https://github.com/yourusername/virtual-lab.git`
2. Open the folder and double-click `index.html` in any modern web browser.
